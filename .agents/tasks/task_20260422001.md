## 需求要点汇总

### 1. 产品图标设计
- 风格：简约几何风格
- 输出：SVG + PNG 多尺寸 + AI Prompt（备选）
- 应用：打包产物、任务栏、主页面 Logo

### 2. 移除默认菜单栏 + 窗口外观
- 移除 Electron 默认菜单栏
- 窗口圆角 16px + 强阴影
- 跨平台兼容

### 3. 首屏错误 Toast 化
- 当前：静态 `error-box` 撑开布局
- 目标：Toast 浮窗，3秒自动消失
- 位置：卡片顶部或屏幕顶部

### 4. 密码库响应式布局
- 当前：固定断点切换（4→3→2→1），跳变明显
- 目标：CSS Grid `auto-fill` 流畅自适应
- 卡片最小宽度 240px

### 5. 新增按钮图标
- 当前：多笔画加号，视觉不够简洁
- 目标：简洁现代，风格统一

### 6. 其他 UI/UX 优化
- 由 UI/UX Agent 审查后补充
