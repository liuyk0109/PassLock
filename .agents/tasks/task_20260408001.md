# 任务：自动锁定功能

**任务ID**：20260408001
**创建时间**：2026-04-08
**状态**：已完成

## 需求简要描述

用户空闲超过配置时间后自动锁定密码库，保护敏感数据安全。在Vue应用中监听用户活动（鼠标、键盘、点击），达到空闲阈值后调用lock()方法清除所有敏感数据。
