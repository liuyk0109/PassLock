## 功能概要

### 核心安全功能

#### 1. AES-256-GCM 加密
- PBKDF2 密钥派生 (100,000 iterations)
- 主密码验证机制
- 密码强度评估

#### 2. 自动锁定功能 (P0)
- 用户空闲超时自动锁定密码库
- 可配置超时时间（设置页面）
- 空闲检测定时器实现

#### 3. 主密码修改功能 (P0)
- 支持更换主密码
- 验证当前密码
- 重新加密所有密码条目
- 更新验证数据

#### 4. 数据备份与恢复功能 (P0)
- 导出加密的密码库数据为JSON文件
- 导入备份恢复数据
- 冲突处理策略（覆盖/跳过/合并）
- 主密码验证导入

#### 5. 密码卡片操作按钮 (P1)
- 编辑按钮：修改密码条目
- 删除按钮：删除条目（带确认对话框）
- 复制按钮：一键复制密码

#### 6. UI优化 (P1)
- 移除默认菜单栏
- Toast组件替代error-box（3秒自动消失）
- 窗口圆角CSS模拟（16px）
- 密码库响应式布局（CSS Grid auto-fill）
- 新增按钮图标SVG更新（简洁单笔画加号）
- 密码生成器功能

## 安装说明

1. 运行 `PassLock Setup 0.1.0-beta.exe`
2. 选择安装目录（支持自定义路径）
3. 完成安装后启动 PassLock
4. 创建主密码以初始化密码库

## 安全建议

- 使用强主密码（至少12位，包含大小写字母、数字、符号）
- 定期备份密码库数据
- 定期修改主密码
- 设置合适的自动锁定超时时间
