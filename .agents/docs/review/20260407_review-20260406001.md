## 严重问题（必须修复）

### 1. ConfirmDialog ESC 键关闭功能失效

**文件**：[ConfirmDialog.vue#L45-L61](/d:/code/PassLock/src/components/ConfirmDialog.vue)

**问题描述**：该组件通过模板中的 `@keydown="handleKeydown"` 绑定键盘事件，但 `.confirm-backdrop` 是一个 `div` 元素，默认不可聚焦。因此，用户按下 ESC 键时，事件监听器不会触发，对话框无法通过 ESC 键关闭。这是一个功能性缺陷，影响用户体验。

**修复方案**：
```vue
<script setup lang="ts">
import { onMounted, onUnmounted } from 'vue'


// 组件挂载时添加键盘监听
onMounted(() => {
  window.addEventListener('keydown', handleKeydown)
})

// 组件卸载时移除键盘监听
onUnmounted(() => {
  window.removeEventListener('keydown', handleKeydown)
})
</script>
```

### 2. VaultPage resize 事件监听器未清理

**文件**：[VaultPage.vue#L121-L128](/d:/code/PassLock/src/components/VaultPage.vue)

**问题描述**：在 `onMounted` 中添加了 resize 事件监听器，但没有在组件卸载时移除。这会导致内存泄漏，组件销毁后事件监听器仍然存在，可能引用已销毁的组件状态。

**修复方案**：
```typescript
import { onMounted, onUnmounted } from 'vue'

// 组件挂载时加载条目
onMounted(async () => {
  await vaultStore.loadEntries()
  
  window.addEventListener('resize', handleResize)
})

// 组件卸载时移除事件监听
onUnmounted(() => {
  window.removeEventListener('resize', handleResize)
})

// resize 处理函数
function handleResize() {
  windowWidth.value = window.innerWidth
}
```

## 重要问题（建议修复）

### 3. lock() 函数未清除剪贴板定时器

**文件**：[vault.ts#L73-L80](/d:/code/PassLock/src/stores/vault.ts)

**问题描述**：锁定密码库时，`lock()` 函数没有清除 `clipboardClearTimer`。如果用户在复制密码后锁定密码库，30秒定时器仍会执行，尝试清除剪贴板。虽然不会导致崩溃，但这不符合预期的安全行为——锁定后应该立即清理所有敏感状态。

**修复方案**：
```typescript
function lock() {
  // 清除剪贴板定时器
  if (clipboardClearTimer.value !== null) {
    clearTimeout(clipboardClearTimer.value)
    clipboardClearTimer.value = null
  }
  
  // 尝试立即清除剪贴板
  navigator.clipboard.writeText('').catch(() => {})
  
  masterKey.value = null
  isLocked.value = true
  entries.value = []
  showModal.value = false
  searchQuery.value = ''
  copiedEntryId.value = null
}
```

### 4. clipboardClearTimer 不应导出

**文件**：[vault.ts#L305](/d:/code/PassLock/src/stores/vault.ts)

**问题描述**：`clipboardClearTimer` 是内部定时器状态，被包含在 store 的导出列表中。这是一个实现细节，外部组件不应该直接访问或修改这个定时器。导出内部状态可能导致意外的状态篡改。

**修复方案**：从导出列表中移除 `clipboardClearTimer`：
```typescript
return {
  // 状态
  isLocked,
  masterKey,
  entries,
  showModal,
  searchQuery,
  copiedEntryId,
  loading,
  // clipboardClearTimer 应移除
  // ...
}
```
