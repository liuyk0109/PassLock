# 密码卡片操作按钮 API 设计规范

## 概述

本文档定义密码卡片编辑、删除功能的组件 Props/Emits API 和状态管理 API。IPC API 复用现有设计，无需新增。

---

## IPC API（复用现有）

### 加密 API

| 方法 | 参数 | 返回 | 用途 |
|------|------|------|------|
| `crypto:encrypt` | plaintext: string, masterKey: string | Promise<string> | 加密新密码 |
| `crypto:decrypt` | encryptedData: string, masterKey: string | Promise<string> | 解密密码供编辑 |

### 数据库 API

| 方法 | 参数 | 返回 | 用途 |
|------|------|------|------|
| `db:updateEntry` | id: string, updates: Partial<VaultEntry> | Promise<boolean> | 更新条目 |
| `db:deleteEntry` | id: string | Promise<boolean> | 删除条目 |

### AddPasswordModal.vue

#### Props 扩展

```typescript

  // 现有
  visible: boolean
  
  // 新增
  mode?: 'add' | 'edit'              // 模式区分，默认 'add'
  entry?: VaultEntry                 // 编辑模式下的条目数据
  decryptedPassword?: string         // 编辑模式下的解密密码
}

// 默认值

  mode: 'add',
  entry: undefined,
  decryptedPassword: undefined
})
```

#### Emits 扩展

```typescript
interface Emits {
  // 现有
  (e: 'close'): void
  (e: 'save', input: NewEntryInput): void
  
  // 新增
  (e: 'update', id: string, input: EditEntryInput): void  // 编辑模式保存
}
```

#### 模式行为差异

| 属性 | add 模式 | edit 模式 |
|------|----------|-----------|
| 标题 | "新增密码条目" | "编辑密码条目" |
| 保存按钮 | "保存" | "更新" |
| 表单初始值 | 空 | 预填充 entry 数据 |
| 密码字段 | 空 | decryptedPassword |
| emit事件 | save | update |
| 关闭行为 | resetForm | resetForm + 清除编辑状态 |

### ConfirmDialog.vue（新增组件）

#### Props

```typescript
interface Props {
  visible: boolean              // 显示状态
  title?: string                // 对话框标题
  message?: string              // 描述文字
  confirmText?: string          // 确认按钮文字
  cancelText?: string           // 取消按钮文字
  danger?: boolean              // 是否危险操作风格（红色按钮）
}

// 默认值
const props = withDefaults(defineProps<Props>(), {
  title: '确定删除？',
  message: '此操作无法撤销。',
  confirmText: '删除',
  cancelText: '取消',
  danger: true
})
```

#### Emits

```typescript
interface Emits {
  (e: 'confirm'): void   // 确认按钮点击
  (e: 'cancel'): void    // 取消按钮点击
}
```

#### 使用示例

```vue
<ConfirmDialog
  :visible="showDeleteConfirm"
  title="确定删除该密码条目吗？"
  message="删除后无法恢复。"
  confirmText="删除"
  cancelText="取消"
  :danger="true"
  @confirm="handleConfirmDelete"
  @cancel="handleCancelDelete"
/>
```

## Pinia Store API

### VaultStore 扩展

#### 新增类型定义

```typescript
// 编辑条目输入类型
export interface EditEntryInput {
  title: string
  username?: string
  password?: string   // 可选，仅修改密码时传入
  site?: string
  url?: string
  notes?: string
}
```

#### 新增方法

```typescript
/**
 * 编辑密码条目
 * @param id 条目ID
 * @param input 编辑输入（仅传入修改的字段）
 * @returns 更新后的条目
 */
async function editEntry(id: string, input: EditEntryInput): Promise<VaultEntry>

/**
 * 获取解密后的密码
 * @param entryId 条目ID
 * @returns 解密后的明文密码
 */
async function getDecryptedPassword(entryId: string): Promise<string>
```

#### 方法实现逻辑

**editEntry 方法**：
```
1. 验证 masterKey 存在
2. 查找条目
3. 如果 input.password 存在，加密新密码
4. 构建 updates 对象
5. 调用 IPC: db:updateEntry
6. 更新本地 entries 状态
7. 返回更新后的条目
```

**getDecryptedPassword 方法**：
```
1. 验证 masterKey 存在
2. 查找条目
3. 调用 IPC: crypto:decrypt
4. 返回明文密码
```
