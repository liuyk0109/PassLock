# 密码库页面 UI设计规范

## 概述

密码库页面是用户解锁主密码后的核心功能页面，用于展示、管理和新增密码条目。设计需延续主密码页面的深色安全风格，采用卡片式布局，简约大气。

---

### 页面层级

```

├────────────────────────────────────────────────────────────────┤

│  [卡片网格区域]                                                │

│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐              │
```

### 尺寸规范

| 区域 | 规格 |
|------|------|
| 页面宽度 | 100vw，最大内容宽度1200px，居中显示 |
| Header高度 | 72px |
| 卡片区域padding | 顶部24px，左右32px（响应式调整） |

## 视觉规范

### 1. 页面背景

延续主密码页面的深色背景风格：

```css
背景渐变: var(--bg-gradient)
/* linear-gradient(145deg, #171717 0%, #262626 50%, #1a1a1a 100%) */

背景装饰: 保留简化版的装饰圆圈动画
- opacity降低至0.02
- 仅保留1-2个装饰圆，避免干扰卡片阅读
```

### 2. Header 区域

```
布局: flex横向排列
  左侧: Logo图标(32×32) + 标题"密码库"
  右侧: 新增按钮

Logo图标:
  - 与LockScreen一致的小锁图标
  - 颜色: var(--primary-500)

标题:
  - 字体: 20px, 600 weight
  - 颜色: var(--text-primary)

新增按钮:
  - 样式: 圆角矩形按钮，与LockScreen的btn-primary风格一致
  - 内容: "+"图标 + "新增"文字
  - 尺寸: padding 10px 20px
  - 背景: var(--primary-gradient)
  - 阴影: 0 4px 14px rgba(20, 184, 166, 0.35)
  - hover: 微微上移(-1px)，阴影增强
```

### 3. 密码卡片

```
卡片容器:
  - 尺寸: 固定 260px × 160px
  - 背景: rgba(38, 38, 38, 0.7)
  - 边框: 1px solid rgba(255, 255, 255, 0.06)
  - 圆角: var(--radius-2xl) (20px)
  - 阴影: 0 4px 20px rgba(0, 0, 0, 0.2)
  - hover阴影: 0 8px 32px rgba(0, 0, 0, 0.3), 边框微微变亮
  - 过渡: 0.25s ease

卡片内部布局 (从上到下):
┌──────────────────────────┐
│ [图标] 名称              │ ← 第一行：应用图标 + 名称
│                          │
│ 网站/应用名              │ ← 第二行：网站或应用标识
│                          │
│ [备注内容...]            │ ← 第三行：备注（可省略）
│                          │
│ ──────────────────────── │ ← 分隔线
│            [复制按钮]    │ ← 底部：操作按钮区
└──────────────────────────┘

卡片内容padding: 16px

名称:
  - 字体: 16px, 600 weight
  - 颜色: var(--text-primary)
  - 单行显示，超长截断...

网站/应用:
  - 字体: 14px, 400 weight
  - 颜色: var(--text-secondary)
  - 单行显示，超长截断...

备注:
  - 字体: 12px, 400 weight
  - 颜色: var(--text-muted)
  - 最大2行显示，超出截断...
  - 省略时显示"—"占位

分隔线:
  - height: 1px
  - background: rgba(255, 255, 255, 0.05)
  - margin: 上下8px

应用图标:
  - 尺寸: 24×24px
  - 默认图标: 钥匙图标
  - 颜色: var(--primary-400)
  - 与名称同行显示，间距8px

复制按钮:
  - 样式: 简约图标按钮
  - 尺寸: 32×32px
  - 圆角: var(--radius-md) (8px)
  - 背景: rgba(255, 255, 255, 0.05)
  - hover背景: var(--primary-500) / rgba(20, 184, 166, 0.2)
  - 图标: 复制图标 (两矩形叠加)
  - 图标尺寸: 16×16px
  - 点击反馈: 转换为"已复制"提示，持续1.5秒
```

### 4. 空状态设计

当密码库无记录时显示：

```
居中显示空状态提示:
  - 图标: 大型保险箱图标(64×64)
  - 图标颜色: var(--text-muted)
  - 文案: "暂无密码条目"
  - 描述: "点击右上角「新增」添加第一条密码"
  - 字体颜色: var(--text-secondary)
```

## 交互行为

### 1. 页面进入动画

```
解锁成功后:
  - LockScreen淡出消失 (opacity 0 → 隐藏, 0.3s)
  - Vault页面淡入显示 (opacity 0 → 1, 0.3s)
  - 卡片依次淡入 (stagger动画，每张延迟50ms)
```

### 2. 卡片交互

```
hover状态:
  - 阴影增强
  - 边框亮度微微提升
  - 背景微微变亮
  - 复制按钮提前显示/高亮

复制按钮点击:
  - 图标切换为"✓"勾选图标
  - 显示提示文字"已复制"
  - 背景变为 success 色调
  - 1.5秒后恢复原状
```

### 3. 新增弹窗交互

```
打开弹窗:
  - 背景模糊动画 (blur 0 → 8px, 0.3s)
  - 弹窗从下方滑入 (translateY 20px → 0, opacity 0 → 1)
  - 自动聚焦到"名称"输入框

关闭弹窗:
  - 点击×按钮关闭
  - 点击背景层关闭
  - 按 ESC 键关闭
  - 弹窗淡出滑下 (反向动画)

保存操作:
  - 验证必填项(名称、密码)
  - 显示loading状态
  - 保存成功后:
    - 弹窗关闭
    - 新卡片添加到列表顶部
    - 新卡片高亮动画(边框闪烁primary色)
  - 保存失败:
    - 底部显示错误提示框
```

### 4. 随机密码生成

```
点击"生成"按钮:
  - 弹出小型下拉面板
  - 面板内容:
    - 密码长度滑块(8-32位)
    - 选项复选框: 大写、小写、数字、符号
    - 生成的密码预览
    - [使用] 按钮
  - 点击"使用"后填充密码输入框
```

| ≥1200px | 4个 | 1200px | 20px |

- 宽屏：Logo + 标题 + 新增按钮(图标+文字)
