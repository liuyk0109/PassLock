# 密码卡片操作按钮 UI设计规范

## 概述

为密码卡片添加编辑、复制、删除操作按钮，采用悬停显示设计以保持界面清爽。按钮排列遵循安全优先原则，危险操作（删除）置于末尾并使用警示色。

---

## 布局结构

### 卡片底部操作区改造

```
现有布局:
┌──────────────────────────┐
│ [图标] 名称              │
│ 网站/应用名              │
│ [备注内容...]            │
│ ──────────────────────── │
│            [复制按钮]    │ ← 单按钮
└──────────────────────────┘

改造后布局:
┌──────────────────────────┐
│ [图标] 名称              │
│ 网站/应用名              │
│ [备注内容...]            │
│ ──────────────────────── │
│  [编辑][复制][删除]      │ ← 三按钮组（hover显示）
└──────────────────────────┘
```

### 操作按钮位置与排列

```
底部操作区:
  - 高度: 32px (与现有复制按钮高度一致)
  - 内容对齐: 右对齐 (justify-content: flex-end)
  - 按钮排列顺序: 编辑 → 复制 → 删除 (从左到右)
  - 按钮间距: 8px (var(--space-sm))
  - 默认状态: 全部按钮隐藏 (opacity: 0)
  - hover状态: 全部按钮显示 (opacity: 1)
```

## 交互行为

### 1. 按钮组悬停显示动画

```
触发条件: 鼠标移入卡片区域 (mouseenter)

动画序列:
  - card-footer opacity: 0 → 1
  - 过渡时间: 200ms (≤200ms 性能要求)
  - 缓动函数: ease-out

离开卡片:
  - card-footer opacity: 1 → 0
  - 过渡时间: 200ms
  - 缓动函数: ease-in

注意: 若复制按钮处于"已复制"状态，不隐藏按钮组
```

### 2. 编辑按钮交互流程

```
点击编辑按钮:
  1. 按钮背景短暂闪烁 (scale效果 0.95 → 1)
  2. 打开编辑弹窗
  3. 弹窗标题显示"编辑密码条目"
  4. 自动填充当前条目数据
  5. 聚焦到名称输入框

编辑弹窗交互:
  - 修改任意字段
  - 点击"更新"保存
  - 或点击"取消"关闭不作修改
  - 支持ESC键关闭

保存成功:
  - 弹窗关闭
  - 卡片内容更新
  - 卡片边框短暂闪烁primary色
```

### 3. 删除按钮交互流程

```
点击删除按钮:
  1. 按钮背景变为红色调
  2. 打开删除确认对话框
  3. 背景模糊效果

确认对话框交互:
  - 点击"取消": 关闭对话框，恢复卡片状态
  - 点击"删除": 执行删除操作

删除执行:
  1. 对话框关闭
  2. 卡片执行离开动画 (card-leave-active)
     - opacity: 1 → 0
     - translateY: 0 → -10px
     - 过渡: 200ms
  3. 卡片从网格中移除
  4. 数据库删除记录
```

### 4. 复制按钮交互流程（保持现有）

```
点击复制按钮:
  1. 执行复制操作
  2. 按钮图标切换为 ✓
  3. 显示"已复制"文字
  4. 背景变为primary色调
  5. 1.5秒后恢复原状

hover状态下复制按钮始终可见
```

## 动画规范

### 按钮组显示动画

```css
/* 卡片hover时按钮组显示 */
.password-card:hover .card-footer {
  opacity: 1;
  transition: opacity 0.2s ease-out;
}

/* 卡片离开hover时按钮组隐藏 */
.password-card:not(:hover) .card-footer {
  opacity: 0;
  transition: opacity 0.2s ease-in;
}
```

### 删除卡片动画（使用现有TransitionGroup）

```css
/* 复用VaultPage现有的卡片离开动画 */
.card-leave-active {
  animation: card-out 0.2s ease;
}

@keyframes card-out {
  from {
    opacity: 1;
    transform: translateY(0);
  }
  to {
    opacity: 0;
    transform: translateY(-10px);
  }
}
```

### 确认对话框动画

```css
/* 对话框进入 */
.confirm-enter-active {
  animation: confirm-in 0.25s ease;
}

@keyframes confirm-in {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}

/* 对话框离开 */
.confirm-leave-active {
  animation: confirm-out 0.2s ease;
}

@keyframes confirm-out {
  from {
    opacity: 1;
    transform: scale(1);
  }
  to {
    opacity: 0;
    transform: scale(0.95);
  }
}
```
