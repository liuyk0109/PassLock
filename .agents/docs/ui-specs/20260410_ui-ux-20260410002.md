# 数据备份与恢复功能 UI设计规范

## 设计概述

本功能为PassLock密码管理器提供数据导出和导入功能，在设置页面（Settings.vue）中新增"数据管理"模块，并新建导入冲突处理弹窗（ImportConflictDialog.vue）。设计遵循现有项目的暗色主题风格，与task_20260410001的设置页面设计保持一致。

### 3. 导入流程交互

#### 导入流程图
```
点击"导入数据" → 弹出系统文件选择对话框（Electron dialog）
    ↓
用户选择备份文件 → 验证文件格式（JSON）
    ↓
格式错误 → 显示错误toast："文件格式不正确"
    ↓
格式正确 → 弹出主密码验证弹窗（输入当前主密码）
    ↓
验证成功 → 解析备份文件，检测条目冲突
    ↓
无冲突 → 直接导入，显示成功toast
    ↓
有冲突 → 弹出ImportConflictDialog，逐条处理
    ↓
用户确认处理 → 执行导入操作
    ↓
导入完成 → 显示成功toast（导入统计）
```

## 交互行为

### 导出功能交互

#### 1. 点击导出按钮
- 触发系统保存对话框（Electron dialog.showSaveDialog）
- 默认文件名：`PassLock_Backup_YYYYMMDD_HHMMSS.json`
- 默认保存位置：用户文档目录或上次保存位置

#### 2. 导出进度
- 导出过程中按钮显示spinner状态（可选）
- 导出操作通常较快（<3秒），进度指示可选

#### 3. 成功反馈
- 关闭系统对话框
- 显示成功toast（顶部居中，3秒后自动消失）
- toast内容："导出成功：{文件名}，共导出 X 条密码条目"

#### 4. 失败反馈
- 显示错误toast："导出失败：{错误原因}"
- 可能的错误：权限不足、磁盘空间不足、路径无效

### 导入功能交互

#### 1. 点击导入按钮
- 触发系统文件选择对话框（Electron dialog.showOpenDialog）
- 文件过滤器：*.json
- 支持多选：否（单文件导入）

#### 2. 文件格式验证
- 选择文件后立即验证JSON格式
- 格式错误 → 显示错误toast："文件格式不正确，请选择PassLock备份文件"

#### 3. 主密码验证弹窗
- 格式正确后弹出主密码验证弹窗
- 输入框：当前主密码（带眼睛图标切换显示）
- 提示文字："密码必须与备份文件的主密码一致才能导入"
- 验证失败 → 显示错误："密码不匹配，无法导入此备份文件"
- 验证成功 → 解析备份文件，检测冲突

#### 4. 冲突检测
- 对比备份条目ID与本地条目ID
- 无冲突 → 直接导入，显示成功toast
- 有冲突 → 弹出ImportConflictDialog

### ImportConflictDialog交互

#### 1. 弹窗打开
- 显示冲突条目数量："检测到 X 条冲突条目"
- 显示无冲突条目数量："无冲突条目：Y 条将自动导入"
- 每条冲突条目默认未选择状态

#### 2. 冲突条目显示
- 本地条目信息：标题 · 用户名 · 创建时间
- 备份条目信息：标题 · 用户名 · 创建时间
- 标记图标：红色圆点（🔴）表示冲突

#### 3. 单条操作选择
- **覆盖**：用备份条目替换本地条目（危险操作）
- **跳过**：保留本地条目，不导入备份条目
- **保留两者**：保留本地条目，备份条目以新ID导入

#### 4. 操作按钮状态
```
未选择状态：
[覆盖]（灰色背景）
[跳过]（灰色背景）
[保留两者]（灰色背景）

已选择状态：
[覆盖]（选中样式，红色高亮）
或 ✓ 已选择：覆盖（文字显示）
```

#### 5. 快捷操作
- **全部覆盖**：所有冲突条目选择"覆盖"
- **全部跳过**：所有冲突条目选择"跳过"
- **全部保留两者**：所有冲突条目选择"保留两者"

#### 6. 确认导入
- 点击"确认导入"按钮
- 未处理的冲突条目默认"跳过"
- 显示进度覆盖层（可选）
- 导入完成后显示成功toast

#### 7. 导入完成反馈
```
成功toast：
┌────────────────────────────────────────────┐
│  ✓  导入成功                               │
│      新增 3 条，覆盖 2 条，跳过 1 条        │
└────────────────────────────────────────────┘
```

#### 8. 关闭弹窗
- 取消按钮：关闭弹窗，不执行导入
- ESC键：关闭弹窗
- 点击背景：关闭弹窗
- 导入完成后：自动关闭弹窗

## 状态定义

### 导出流程状态

| 状态 | 描述 | UI表现 |
|------|------|--------|
| idle | 待触发 | 导出按钮正常显示 |
| selecting | 选择保存位置 | 系统保存对话框打开 |
| exporting | 导出中 | 按钮显示spinner（可选） |
| success | 导出成功 | 显示成功toast |
| error | 导出失败 | 显示错误toast |

### 导入流程状态

| 状态 | 描述 | UI表现 |
|------|------|--------|
| idle | 待触发 | 导入按钮正常显示 |
| selecting | 选择备份文件 | 系统文件选择对话框打开 |
| verifying | 验证主密码 | 显示主密码验证弹窗 |
| checking | 检测冲突 | 显示loading状态 |
| conflict | 存在冲突 | 显示ImportConflictDialog |
| importing | 导入中 | 显示进度覆盖层（可选） |
| success | 导入成功 | 显示成功toast |
| error | 导入失败 | 显示错误toast |

### 冲突条目状态

| 状态 | 描述 | UI表现 |
|------|------|--------|
| pending | 未处理 | 显示三个操作按钮 |
| override | 选择覆盖 | 显示"已选择：覆盖" |
| skip | 选择跳过 | 显示"已选择：跳过" |
| keepboth | 选择保留两者 | 显示"已选择：保留两者" |
